# Motus — Real-Time Back Pain Relief at Your Desk

Motus is your always-available desktop companion for instant back pain relief. This is the landing page for the Motus desktop application.

## Features

- **Interactive 3D Body Model** — Pinpoint exactly where you're hurting
- **Personalized Protocols** — Get targeted stretches for your specific pain point
- **Cognitive Calming** — Breathing exercises to manage pain response
- **Smart Nudges** — Proactive reminders to prevent pain before it builds

## Tech Stack

- **Next.js 15** — React framework with App Router
- **Tailwind CSS** — Utility-first CSS
- **Framer Motion** — Smooth animations
- **Lucide React** — Beautiful icons
- **TypeScript** — Type safety

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Project Structure

```
src/
├── app/
│   ├── globals.css    # Global styles and CSS variables
│   ├── layout.tsx     # Root layout with fonts and metadata
│   └── page.tsx       # Main landing page
```
